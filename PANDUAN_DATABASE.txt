PANDUAN DATABASE BAPENDA

Nama database:
bapenda

File SQL:
C:\xampp\htdocs\bapenda\database.sql

Cara import lewat phpMyAdmin:
1. Buka http://localhost/phpmyadmin
2. Klik menu Import
3. Pilih file C:\xampp\htdocs\bapenda\database.sql
4. Klik Go / Kirim
5. Setelah berhasil, database bapenda akan berisi tabel:
   - admin
   - pengajuan
   - berkas

Cara import lewat terminal MySQL XAMPP:
C:\xampp\mysql\bin\mysql.exe -u root < C:\xampp\htdocs\bapenda\database.sql

Perintah cek database:
C:\xampp\mysql\bin\mysql.exe -u root -e "SHOW DATABASES;"

Perintah cek tabel:
C:\xampp\mysql\bin\mysql.exe -u root -e "USE bapenda; SHOW TABLES;"

Perintah cek data pengajuan:
C:\xampp\mysql\bin\mysql.exe -u root -e "USE bapenda; SELECT * FROM pengajuan;"

Akun admin contoh:
Username: admin
Password: admin123

Alur data:
1. Masyarakat mengisi form di http://localhost/bapenda/form_pengajuan.php
2. Data utama masuk ke tabel pengajuan
3. Nama file KTP dan dokumen pendukung masuk ke tabel berkas
4. File fisik tersimpan di folder C:\xampp\htdocs\bapenda\uploads
5. Admin melihat data di http://localhost/bapenda/admin/dashboard.php
